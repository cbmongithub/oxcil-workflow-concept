# OXCIL — Baseline Reference Pack

This folder contains the initial baseline reference documents for Oxcil.

## Files

- `OXCIL_ARCHITECTURE_BASELINE.md` — detailed product and architecture definition
- `OXCIL_V1_SCOPE.md` — V1 scope, features, boundaries, and launch criteria
- `OXCIL_ROADMAP.md` — phased roadmap after internal validation and V1

## Purpose

These documents are intended to be living references used to:

- align product direction
- guide implementation
- onboard future contributors or agents
- keep Oxcil focused as scope expands

## Theme direction

Oxcil should use a dark, Vercel-like visual language:

- black / near-black surfaces
- subtle OKLCH borders
- restrained monochrome UI
- high trust, system-first design
